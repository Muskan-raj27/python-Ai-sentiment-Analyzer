# 🤖 AI Sentiment Analyzer

## 📖 Project Overview

AI Sentiment Analyzer is a Python-based application that uses Natural Language Processing (NLP) to analyze the sentiment of user-provided text. It classifies the input as **Positive**, **Negative**, or **Neutral** using the **TextBlob** library.

## 🎯 Project Scope

This project aims to:
- Analyze the sentiment of user-entered text.

- Demonstrate the basics of Natural Language Processing (NLP).
- Build a beginner-friendly AI project using Python.
- Understand how sentiment analysis is used in real-world applications.

---

## ✨ Features

- Analyze text sentiment instantly.
- Displays sentiment polarity score.
- Classifies text as Positive, Negative, or Neutral.
- Simple command-line interface.
- Easy to understand and beginner-friendly.
